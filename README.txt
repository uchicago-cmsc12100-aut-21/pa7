CS 121 Programming Assignment #7: Analyzing police traffic stop data
--------------------------------------------------------------------

These are the files for Programming Assignment #7.

- traffic_stops.py: Python file where you will write your code.

- pa7_helpers.py: Python file with some helper functions

- test_traffic_stops.py: Python file with the automated tests for this assignment.

- get_files.sh: A script for downloading the data. See the programming 
  assignment writeup for instructions on how to run it. Running it will add
  a data/ directory.

- pytest.ini: A configuration file that you can safely ignore.

- README.txt: this file.
